- Copy from P. Wood template, OR
- Download from Pexels.com (search "electrician"), OR
- Skip entirely (gradient backgrounds look professional)

### 4. Deploy
```bash
# Use the deployment script or manually copy files to target location
bash deploy-demos.sh
```
